00) Initialize project with package.json: npm init -y
01) Install webpack: npm install webpack --save-dev
02) Install JQuery: npm install jquery --save-dev
03) Run webpack to create bundle: npm run wbp
04) Install Babel: npm install babel-loader@8.0.0-beta.0 --save-dev
	npm install @babel/core@7.0.0-beta.31 --save-dev
	npm install @babel/preset-env@7.0.0-beta.31 --save-dev
05) Install Bootstrap: npm install popper.js --save-dev
	npm install bootstrap@4.0.0-beta.2 --save-dev
06) Install CSS loader: npm install css-loader --save-dev
07) Install style loader: npm install style-loader --save-dev
